Notes & Decisions
Player Rating Formula
Confirmed Mar 9, 2026. Formula: Points Earned ÷ Points Available × 100. Singles win = 3 pts, Doubles win = 2 pts, any loss or appearance = 0 pts. Replaced old formula that awarded 1 pt for appearance — players were unhappy. Phase 2: formula becomes configurable per competition.
Fixed Matchdays — Deferred
Moved to Phase 2 (Mar 9, 2026). Fixed Matchday scheduling (central venue, multiple games per day) is typically used in elimination-style events, not round-robin leagues. Weekly rounds covers all MVP use cases.
⚠️ Critical Tech Debt
Current DB uses a key-value store (JSONB blobs in a single table). No transactions, no indexing, no referential integrity. Must migrate to proper relational schema before any public launch. This is the #1 pre-launch blocker.
Venue Conflict Logic — MVP
When two teams share the same home venue, the match generator must ensure one team plays away on that matchday. This is part of the MVP scheduling logic, not deferred. Important for leagues using home/away venue mode.
